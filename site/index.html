Quick Start
# Install
npm install -g solanatestforge

# Fork devnet state
solforge fork devnet <PROGRAM_ID>

# Run security scan
solforge security ./target/idl/my_program.json

# Fuzz test
solforge fuzz ./target/idl/my_program.json
